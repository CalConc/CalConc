Visualizar Agregado
-------------------

Mostra as informações do agregado selecionado.

Campos vazios não tem valor.

As informações visualizadas são:

    - Nome
    - Cidade
    - Bairro
    - Logradouro
    - CEP
    - Complemento
    - CPF/CNPJ
    - Fone 1
    - Fone 2
    - IE
    - Observação
    - Data de Cadastro
