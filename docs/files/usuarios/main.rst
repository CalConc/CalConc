Usuários
--------

A tela inicial de usuário mostra todos os usuários cadastrados, dando destaque às informações 'Nome', 'Email', 'Usuário', 'Grupo' e 'Status.

Novos usuários podem ser cadastrado clicando em 'Adicionar Usuário'

Usuários já cadastrados podem ser visualizados (ícone de olho na coluna 'Ações'), editados (ícone de lapis na coluna 'Ações') e desativados (ícone de xis na coluna 'Ações'), caso o usuário esteja ativo, ativados (ícone de certo na coluna 'Ações'), caso o usuário esteja desativado.

Usuários inativados não podem realizar nenhuma ação no sistema, inclusive autenticação.
